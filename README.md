  ## Usage/Installation
  You can install this app by cloning it onto your copmuter, after that you will need to open the terminal at the root folder of the project and run npm install or npm i, then you are ready to go.

  After following the installation steps above, you can just run node app in the console, at the root of the folder, and answer all the prompts. After that the files for the portfolio will be created at the dist directory
